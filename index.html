Programmazione C

Questa pagina presenta il materiale didattico per il corso di Programmazione C di Fausto Spoto presso il Dipartimento di Informatica dell'Università di Verona. Per copiare l'intero materiale didattico sulla vostra macchina locale, istallate git ed eseguite il comando:

$ git clone git@github.com:spoto/ProgrammazioneC.git

Compiti e Parziali

Due directory contengono i compiti degli ultimi anni, sia i completi che i parziali.

Diario Didattico

Le lezioni sono organizzate in ordine progressivo e contengono il materiale didattico discusso in classe.
